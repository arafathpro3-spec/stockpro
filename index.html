🔥 Connexion Firebase (gratuit)
Firebase sauvegarde vos données dans le cloud gratuitement.
1. Allez sur console.firebase.google.com
2. Créez un projet → Realtime Database → Créer
3. Copiez l'URL (ex: https://mon-projet-rtdb.firebaseio.com)
4. Dans les règles, mettez "read": true, "write": true
⚠️ Retenez bien ce nom — il vous servira à récupérer vos données
